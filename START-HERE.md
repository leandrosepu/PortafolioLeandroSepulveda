## 🎨 Customization Highlights

### Update Contact Info
```javascript
// portfolio.jsx around line 1200
email: 'your@email.com'
github: 'yourgithub'
linkedin: 'yourlinkedin'
phone: '+56 9 XXXX XXXX'
```

### Add Projects
```javascript
// Find ProjectsSection in portfolio.jsx
const projects = [
  {
    title: 'Your Project',
    desc: 'What you built',
    tech: ['React', 'Node.js'],
    status: 'Complete',
    github: 'https://github.com/you/project',
    live: 'https://your-project.com',
  },
];
```

### Change Colors
```javascript
// tailwind.config.js
cyan: {
  400: '#0ea5e9',  // ← Change these
  500: '#06b6d4',
}
```
